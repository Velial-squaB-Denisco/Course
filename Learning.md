**–®–∞–≥ 1: –ë–∏–Ω–æ–º–∏–∞–ª—å–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ß–∏—Å–ª–æ —É—Å–ø–µ—Ö–æ–≤ –≤ `n` –Ω–µ–∑–∞–≤–∏—Å–∏–º—ã—Ö –∏—Å–ø—ã—Ç–∞–Ω–∏—è—Ö —Å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é `p`.  
**–§–æ—Ä–º—É–ª–∞**:  
\[ P(X = k) = C(n, k) \times p^k \times (1 - p)^{n - k} \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã–ø–∞–¥–µ–Ω–∏—è 3 –æ—Ä–ª–æ–≤ –≤ 5 –±—Ä–æ—Å–∫–∞—Ö:  
  \[ P(X = 3) = C(5, 3) \times 0.5^3 \times 0.5^2 = 10 \times 0.125 \times 0.25 = 0.3125. \]  

**–ö–æ–¥**:  
```python
from scipy.stats import binom
n, p = 5, 0.5
print(f"P(X=3): {binom.pmf(3, n, p):.4f}")  # 0.3125
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø–æ–ª—É—á–∏—Ç—å 2 —à–µ—Å—Ç—ë—Ä–∫–∏ –ø—Ä–∏ 4 –±—Ä–æ—Å–∫–∞—Ö –∫—É–±–∏–∫–∞? (–û—Ç–≤–µ—Ç: ‚âà 0.1157).

**–®–∞–≥ 2: –†–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –ü—É–∞—Å—Å–æ–Ω–∞**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ú–æ–¥–µ–ª–∏—Ä—É–µ—Ç —Ä–µ–¥–∫–∏–µ —Å–æ–±—ã—Ç–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, —á–∏—Å–ª–æ –∑–≤–æ–Ω–∫–æ–≤ –≤ –∫–æ–ª-—Ü–µ–Ω—Ç—Ä –∑–∞ —á–∞—Å).  
**–§–æ—Ä–º—É–ª–∞**:  
\[ P(X = k) = \frac{\lambda^k \times e^{-\lambda}}{k!} \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –ï—Å–ª–∏ –≤ –º–∞–≥–∞–∑–∏–Ω –≤ —Å—Ä–µ–¥–Ω–µ–º –ø—Ä–∏—Ö–æ–¥–∏—Ç 5 –∫–ª–∏–µ–Ω—Ç–æ–≤ –≤ —á–∞—Å, –∫–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø—Ä–∏—Ö–æ–¥–∞ 3 –∫–ª–∏–µ–Ω—Ç–æ–≤?  
  \[ P(X = 3) = \frac{5^3 \times e^{-5}}{3!} ‚âà 0.1404. \]  

**–ö–æ–¥**:  
```python
from scipy.stats import poisson
lambda_param = 5
print(f"P(X=3): {poisson.pmf(3, lambda_param):.4f}")  # 0.1404
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ï—Å–ª–∏ Œª = 2, –Ω–∞–π–¥–∏—Ç–µ P(X=0). (–û—Ç–≤–µ—Ç: ‚âà 0.1353).

**–®–∞–≥ 3: –ì–∏–ø–µ—Ä–≥–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã–±—Ä–∞—Ç—å `k` —É—Å–ø–µ—Ö–æ–≤ –±–µ–∑ –≤–æ–∑–≤—Ä–∞—â–µ–Ω–∏—è.  
**–§–æ—Ä–º—É–ª–∞**:  
\[ P(X = k) = \frac{C(K, k) \times C(N-K, n-k)}{C(N, n)} \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –í –∫–æ–ª–æ–¥–µ 4 —Ç—É–∑–∞. –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã—Ç–∞—â–∏—Ç—å 2 —Ç—É–∑–∞ –∏–∑ 5 –∫–∞—Ä—Ç?  
  \[ P(X=2) = \frac{C(4, 2) \times C(48, 3)}{C(52, 5)} ‚âà 0.0399. \]  

**–ö–æ–¥**:  
```python
from scipy.stats import hypergeom

```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –í –∫–æ—Ä–∑–∏–Ω–µ 10 —è–±–ª–æ–∫, 3 –≥–Ω–∏–ª—ã—Ö. –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã–±—Ä–∞—Ç—å 1 –≥–Ω–∏–ª–æ–µ –∏–∑ 4? (–û—Ç–≤–µ—Ç: ‚âà 0.5000).

#### **–£—Ä–æ–∫ 2.2: –ù–µ–ø—Ä–µ—Ä—ã–≤–Ω—ã–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è ‚Äî –≤—Ä–µ–º—è, —Ä–æ—Å—Ç –∏ –æ—à–∏–±–∫–∏**  
**–¶–µ–ª—å**: –ü–æ–Ω—è—Ç—å, –∫–∞–∫ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω—ã –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω—ã–µ –≤–µ–ª–∏—á–∏–Ω—ã (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ä–æ—Å—Ç –ª—é–¥–µ–π).  

---

**–®–∞–≥ 1: –†–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –í—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤ –∏–Ω—Ç–µ—Ä–≤–∞–ª–µ [a, b] —Ä–∞–≤–Ω–æ–≤–µ—Ä–æ—è—Ç–Ω—ã.  
**–§–æ—Ä–º—É–ª–∞ –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏**:  
\[ f(x) = \frac{1}{b - a} \text{ –¥–ª—è } x \in [a, b]. \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –ê–≤—Ç–æ–±—É—Å –ø—Ä–∏—Ö–æ–¥–∏—Ç –∫–∞–∂–¥—ã–µ 10 –º–∏–Ω—É—Ç. –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∂–¥–∞—Ç—å –º–µ–Ω—å—à–µ 3 –º–∏–Ω—É—Ç?  
  **–†–µ—à–µ–Ω–∏–µ**: –ü–ª–æ—â–∞–¥—å –ø—Ä—è–º–æ—É–≥–æ–ª—å–Ω–∏–∫–∞ [0, 3] –≤—ã—Å–æ—Ç–æ–π 0.1: 3 √ó 0.1 = 0.3.  

**–ö–æ–¥**:  
```python
from scipy.stats import uniform
a, b = 0, 10
print(f"P(X < 3): {uniform.cdf(3, a, b):.2f}")  # 0.30
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ï—Å–ª–∏ –≤—Ä–µ–º—è –æ–∂–∏–¥–∞–Ω–∏—è —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ –Ω–∞ [0, 20], –Ω–∞–π–¥–∏—Ç–µ P(5 < X < 15). (–û—Ç–≤–µ—Ç: 0.5).

**–®–∞–≥ 2: –ù–æ—Ä–º–∞–ª—å–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ö–æ–ª–æ–∫–æ–ª–æ–æ–±—Ä–∞–∑–Ω–∞—è –∫—Ä–∏–≤–∞—è, –æ–ø–∏—Å—ã–≤–∞—é—â–∞—è –º–Ω–æ–≥–∏–µ –ø—Ä–∏—Ä–æ–¥–Ω—ã–µ —è–≤–ª–µ–Ω–∏—è.  
**–§–æ—Ä–º—É–ª–∞ –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏**:  
\[ f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}} \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –†–æ—Å—Ç –º—É–∂—á–∏–Ω: Œº = 175 —Å–º, œÉ = 10 —Å–º.  
- –ö–∞–∫–æ–π –ø—Ä–æ—Ü–µ–Ω—Ç –º—É–∂—á–∏–Ω –≤—ã—à–µ 190 —Å–º?  
  **–†–µ—à–µ–Ω–∏–µ**:  
  - 190 = Œº + 1.5œÉ.  
  - –ü–æ –ø—Ä–∞–≤–∏–ª—É 68-95-99.7, –≤—ã—à–µ Œº + 2œÉ ‚Äî 2.5%, –Ω–æ —Ç–æ—á–Ω–µ–µ —á–µ—Ä–µ–∑ Z-—Ç–∞–±–ª–∏—Ü—ã: ‚âà 6.68%.  

**–ö–æ–¥**:  
```python
from scipy.stats import norm

prob = 1 - norm.cdf(190, mu, sigma)
print(f"–ü—Ä–æ—Ü–µ–Ω—Ç –≤—ã—à–µ 190 —Å–º: {prob:.4f}")  # ‚âà 0.0668
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ï—Å–ª–∏ IQ ~ N(100, 15), –Ω–∞–π–¥–∏—Ç–µ P(85 < X < 115). (–û—Ç–≤–µ—Ç: ‚âà 68%).

**–®–∞–≥ 3: –≠–∫—Å–ø–æ–Ω–µ–Ω—Ü–∏–∞–ª—å–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ú–æ–¥–µ–ª–∏—Ä—É–µ—Ç –≤—Ä–µ–º—è –º–µ–∂–¥—É —Å–æ–±—ã—Ç–∏—è–º–∏ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∑–≤–æ–Ω–∫–∞–º–∏).  
**–§–æ—Ä–º—É–ª–∞ –ø–ª–æ—Ç–Ω–æ—Å—Ç–∏**:  
\[ f(x) = \lambda e^{-\lambda x} \text{ –¥–ª—è } x \geq 0. \]  

**–ü—Ä–∏–º–µ—Ä**:  
- –°—Ä–µ–¥–Ω–µ–µ –≤—Ä–µ–º—è –º–µ–∂–¥—É –∑–≤–æ–Ω–∫–∞–º–∏ ‚Äî 10 –º–∏–Ω—É—Ç (Œª = 1/10).  
- –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ —Å–ª–µ–¥—É—é—â–∏–π –∑–≤–æ–Ω–æ–∫ —á–µ—Ä–µ–∑ 5 –º–∏–Ω—É—Ç?  
  **–†–µ—à–µ–Ω–∏–µ**:  
  \[ P(X > 5) = e^{-5/10} ‚âà 0.6065. \]  

**–ö–æ–¥**:  
```python

print(f"P(X > 5): {expon.sf(5, scale=1/lambda_param):.4f}")  # 0.6065
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ï—Å–ª–∏ Œª = 0.2, –Ω–∞–π–¥–∏—Ç–µ P(X < 3). (–û—Ç–≤–µ—Ç: ‚âà 0.4512).

### **–£—Ä–æ–≤–µ–Ω—å 3: –°—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ ‚Äî –∞–Ω–∞–ª–∏–∑ –¥–∞–Ω–Ω—ã—Ö**  
#### **–£—Ä–æ–∫ 3.1: –û–ø–∏—Å–∞—Ç–µ–ª—å–Ω–∞—è —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞ ‚Äî —á–∏—Å–ª–∞ —Ä–∞—Å—Å–∫–∞–∑—ã–≤–∞—é—Ç –∏—Å—Ç–æ—Ä–∏—é**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –æ–ø–∏—Å—ã–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –Ω–µ—Å–∫–æ–ª—å–∫–∏–º–∏ —á–∏—Å–ª–∞–º–∏.

**–®–∞–≥ 1: –ú–µ—Ä—ã —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω–æ–π —Ç–µ–Ω–¥–µ–Ω—Ü–∏–∏**  
- **–°—Ä–µ–¥–Ω–µ–µ**: \(\bar{x} = \frac{1}{n} \sum x_i\).  
- **–ú–µ–¥–∏–∞–Ω–∞**: –°—Ä–µ–¥–Ω–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –≤ —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ–º —Ä—è–¥—É.  
- **–ú–æ–¥–∞**: –ù–∞–∏–±–æ–ª–µ–µ —á–∞—Å—Ç–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.  

**–ü—Ä–∏–º–µ—Ä**:

  - –°—Ä–µ–¥–Ω–µ–µ: (2 + 3 + 5 + 7 + 11)/5 = 5.6.  
  - –ú–µ–¥–∏–∞–Ω–∞: 5.  
  - –ú–æ–¥–∞: –Ω–µ—Ç.  

**–ö–æ–¥**:  

- –î–ª—è –¥–∞–Ω–Ω—ã—Ö [1, 2, 2, 3, 4] –Ω–∞–π–¥–∏—Ç–µ –º–æ–¥—É. (–û—Ç–≤–µ—Ç: 2).

**–®–∞–≥ 2: –ú–µ—Ä—ã —Ä–∞–∑–±—Ä–æ—Å–∞**  
- **–î–∏—Å–ø–µ—Ä—Å–∏—è**: \(\sigma^2 = \frac{1}{n} \sum (x_i - \bar{x})^2\).

- **–ú–µ–∂–∫–≤–∞—Ä—Ç–∏–ª—å–Ω—ã–π —Ä–∞–∑–º–∞—Ö (IQR)**: Q3 - Q1.

  - –°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–µ –æ—Ç–∫–ª–æ–Ω–µ–Ω–∏–µ: \(\sqrt{5.25} ‚âà 2.29\).

#### **–£—Ä–æ–∫ 3.2: –ü—Ä–æ–≤–µ—Ä–∫–∞ –≥–∏–ø–æ—Ç–µ–∑ ‚Äî –∫–∞–∫ –ø—Ä–∏–Ω–∏–º–∞—Ç—å —Ä–µ—à–µ–Ω–∏—è –Ω–∞ –æ—Å–Ω–æ–≤–µ –¥–∞–Ω–Ω—ã—Ö**

- p = 0.03 ‚Üí –û—Ç–≤–µ—Ä–≥–∞–µ–º H‚ÇÄ –Ω–∞ —É—Ä–æ–≤–Ω–µ –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏ 5%.

**–®–∞–≥ 3: t-—Ç–µ—Å—Ç –¥–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ä–µ–¥–Ω–∏—Ö**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, —Ä–∞–∑–ª–∏—á–∞—é—Ç—Å—è –ª–∏ —Å—Ä–µ–¥–Ω–∏–µ –¥–≤—É—Ö –≥—Ä—É–ø–ø.  
**–ü—Ä–∏–º–µ—Ä**:  
- –ì—Ä—É–ø–ø–∞ A: [5, 6, 7, 8, 9] (—Å—Ä–µ–¥–Ω–µ–µ = 7).  
- –ì—Ä—É–ø–ø–∞ B: [3, 4, 5, 6, 7] (—Å—Ä–µ–¥–Ω–µ–µ = 5).  
- –ï—Å—Ç—å –ª–∏ —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏ –∑–Ω–∞—á–∏–º–∞—è —Ä–∞–∑–Ω–∏—Ü–∞?  

**–ö–æ–¥**:

### **–£—Ä–æ–≤–µ–Ω—å 4: –ë–∞–π–µ—Å–æ–≤—Å–∫–∞—è —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∞**  
#### **–£—Ä–æ–∫ 4.1: –ë–∞–π–µ—Å–æ–≤—Å–∫–∏–π –≤—ã–≤–æ–¥ ‚Äî –∫–∞–∫ –æ–±–Ω–æ–≤–ª—è—Ç—å —É–±–µ–∂–¥–µ–Ω–∏—è**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∞–ø—Ä–∏–æ—Ä–Ω—ã–µ –∑–Ω–∞–Ω–∏—è –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞ –¥–∞–Ω–Ω—ã—Ö.

**–®–∞–≥ 1: –§–æ—Ä–º—É–ª–∞ –ë–∞–π–µ—Å–∞**  
\[ P(A|B) = \frac{P(B|A) \times P(A)}{P(B)} \] 
- **P(A)**: –ê–ø—Ä–∏–æ—Ä–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å.  
- **P(B|A)**: –ü—Ä–∞–≤–¥–æ–ø–æ–¥–æ–±–∏–µ.  
- **P(A|B)**: –ê–ø–æ—Å—Ç–µ—Ä–∏–æ—Ä–Ω–∞—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å.  

**–ü—Ä–∏–º–µ—Ä**:  
- –ë–æ–ª–µ–∑–Ω—å –µ—Å—Ç—å —É 1% –ª—é–¥–µ–π. –¢–µ—Å—Ç —Ç–æ—á–Ω–æ—Å—Ç—å 95%. –ü—Ä–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–º —Ç–µ—Å—Ç–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –±–æ–ª–µ–∑–Ω–∏ ‚âà 16%.  

---

**–®–∞–≥ 2: –°–æ–ø—Ä—è–∂–µ–Ω–Ω—ã–µ –∞–ø—Ä–∏–æ—Ä—ã**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ê–ø—Ä–∏–æ—Ä –∏ –∞–ø–æ—Å—Ç–µ—Ä–∏–æ—Ä –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–∞—Ç –æ–¥–Ω–æ–º—É –∫–ª–∞—Å—Å—É —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–π.  
- –ë–µ—Ç–∞-—Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –¥–ª—è –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏.  
- –ì–∞–º–º–∞-—Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –¥–ª—è –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç–∏.  

**–ü—Ä–∏–º–µ—Ä**:  
- –ê–ø—Ä–∏–æ—Ä: Beta(Œ±=1, Œ≤=1) ‚Üí –†–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ.  
- –î–∞–Ω–Ω—ã–µ: 7 —É—Å–ø–µ—Ö–æ–≤ –∏–∑ 10 –∏—Å–ø—ã—Ç–∞–Ω–∏–π.  
- –ê–ø–æ—Å—Ç–µ—Ä–∏–æ—Ä: Beta(Œ±=8, Œ≤=4).  

**–ö–æ–¥**:

#### **–£—Ä–æ–∫ 4.2: MCMC ‚Äî –∫–∞–∫ –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞—Ç—å —Å–ª–æ–∂–Ω—ã–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è —Ä–∞–±–æ—Ç–∞—Ç—å —Å –Ω–µ—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–º–∏ –º–æ–¥–µ–ª—è–º–∏.  

---

**–®–∞–≥ 1: –ê–ª–≥–æ—Ä–∏—Ç–º –ú–µ—Ç—Ä–æ–ø–æ–ª–∏—Å–∞-–ì–∞—Å—Ç–∏–Ω–≥—Å–∞**  
1. –í—ã–±–µ—Ä–∏—Ç–µ –Ω–∞—á–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ \(x_0\).  
2. –ü—Ä–µ–¥–ª–æ–∂–∏—Ç–µ –Ω–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ \(x'\) –∏–∑ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è-–∫–∞–Ω–¥–∏–¥–∞—Ç–∞.  
3. –ü—Ä–∏–º–∏—Ç–µ \(x'\) —Å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å—é \(\min\left(1, \frac{P(x')}{P(x)}\right)\).  

**–ü—Ä–∏–º–µ—Ä**:  
- –û—Ü–µ–Ω–∫–∞ —Å—Ä–µ–¥–Ω–µ–≥–æ –Ω–æ—Ä–º–∞–ª—å–Ω–æ–≥–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è.  

---

**–®–∞–≥ 2: –†–µ–∞–ª–∏–∑–∞—Ü–∏—è –Ω–∞ Python**  
```python
import numpy as np
from scipy.stats import norm

# –¶–µ–ª–µ–≤–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ: N(3, 2)
def target(x):
    return norm.pdf(x, loc=3, scale=2)

# –ê–ª–≥–æ—Ä–∏—Ç–º –ú–µ—Ç—Ä–æ–ø–æ–ª–∏—Å–∞
samples = []
current = 0.0
for _ in range(10000):
    proposal = current + np.random.normal(0, 1)
    acceptance_ratio = target(proposal) / target(current)
    if np.random.rand() < acceptance_ratio:
        current = proposal
    samples.append(current)

print(f"–°—Ä–µ–¥–Ω–µ–µ: {np.mean(samples):.2f}")  # ‚âà 3.0
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ò–∑–º–µ–Ω–∏—Ç–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –Ω–æ—Ä–º–∞–ª—å–Ω–æ–≥–æ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –∏ –ø—Ä–æ–≤–µ—Ä—å—Ç–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç.

### **–£—Ä–æ–≤–µ–Ω—å 5: –ú–∞—à–∏–Ω–Ω–æ–µ –æ–±—É—á–µ–Ω–∏–µ**  
#### **–£—Ä–æ–∫ 5.1: –õ–∏–Ω–µ–π–Ω–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏—è ‚Äî –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ —á–∏—Å–µ–ª**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –ø—Ä–µ–¥—Å–∫–∞–∑—ã–≤–∞—Ç—å —Ü–µ–Ω—ã, –ø—Ä–æ–¥–∞–∂–∏ –∏ –¥—Ä—É–≥–∏–µ –≤–µ–ª–∏—á–∏–Ω—ã.  

---

**–®–∞–≥ 1: –§–æ—Ä–º—É–ª–∞ –ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏**  
\[ y = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + ... + \beta_n x_n + \epsilon \]  
- **–¶–µ–ª—å**: –ù–∞–π—Ç–∏ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã \(\beta\), –º–∏–Ω–∏–º–∏–∑–∏—Ä—É—é—â–∏–µ –æ—à–∏–±–∫—É.  

**–ü—Ä–∏–º–µ—Ä**:  
- –ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ —Ü–µ–Ω—ã –¥–æ–º–∞ –Ω–∞ –æ—Å–Ω–æ–≤–µ –ø–ª–æ—â–∞–¥–∏.  

**–ö–æ–¥**:

```python
model = LinearRegression()
model.fit(X, y)
print(f"–ö–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç: {model.coef_[0]:.2f}")  # 3.0 (—Ü–µ–Ω–∞ –∑–∞ –º¬≤)
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –î–æ–±–∞–≤—å—Ç–µ –ø—Ä–∏–∑–Ω–∞–∫ "–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–æ–º–Ω–∞—Ç" –∏ –æ–±—É—á–∏—Ç–µ –º–æ–¥–µ–ª—å.  

---

#### **–£—Ä–æ–∫ 5.2: –ö–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ü–∏—è ‚Äî –ª–æ–≥–∏—Å—Ç–∏—á–µ—Å–∫–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏—è**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –ø—Ä–µ–¥—Å–∫–∞–∑—ã–≤–∞—Ç—å –∫–∞—Ç–µ–≥–æ—Ä–∏–∏ (—Å–ø–∞–º/–Ω–µ —Å–ø–∞–º).  

---

**–®–∞–≥ 1: –°–∏–≥–º–æ–∏–¥–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è**  
\[ P(y=1) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 x)}} \]  
- –í—ã–≤–æ–¥–∏—Ç –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –ø—Ä–∏–Ω–∞–¥–ª–µ–∂–Ω–æ—Å—Ç–∏ –∫ –∫–ª–∞—Å—Å—É 1.  

**–ü—Ä–∏–º–µ—Ä**:  
- –ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ –¥–µ—Ñ–æ–ª—Ç–∞ –ø–æ –∫—Ä–µ–¥–∏—Ç—É.  

**–ö–æ–¥**:  
```python

X = [[2000], [3000], [4000]]  # –ó–∞—Ä–ø–ª–∞—Ç–∞

print(f"–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –¥–µ—Ñ–æ–ª—Ç–∞ –¥–ª—è 2500$: {model.predict_proba([[2500]])[0][1]:.2f}")
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –î–æ–±–∞–≤—å—Ç–µ –ø—Ä–∏–∑–Ω–∞–∫ "–≤–æ–∑—Ä–∞—Å—Ç" –∏ —Å—Ä–∞–≤–Ω–∏—Ç–µ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã.  

---

### **–£—Ä–æ–≤–µ–Ω—å 6: –≠–∫—Å–ø–µ—Ä—Ç–Ω—ã–µ —Ç–µ–º—ã**  
#### **–£—Ä–æ–∫ 6.1: –°—Ç–æ—Ö–∞—Å—Ç–∏—á–µ—Å–∫–∏–µ –ø—Ä–æ—Ü–µ—Å—Å—ã ‚Äî –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–ª—É—á–∞–π–Ω–æ—Å—Ç–∏ –≤–æ –≤—Ä–µ–º–µ–Ω–∏**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –ø—Ä–µ–¥—Å–∫–∞–∑—ã–≤–∞—Ç—å –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–µ —Å–∏—Å—Ç–µ–º—ã (—Ü–µ–Ω—ã –∞–∫—Ü–∏–π, –ø–æ–≥–æ–¥—É).  

---

**–®–∞–≥ 1: –ë—Ä–æ—É–Ω–æ–≤—Å–∫–æ–µ –¥–≤–∏–∂–µ–Ω–∏–µ**  
**–ü–æ–Ω—è—Ç–∏–µ**: –°–ª—É—á–∞–π–Ω–æ–µ –±–ª—É–∂–¥–∞–Ω–∏–µ —Å –Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω—ã–º –≤—Ä–µ–º–µ–Ω–µ–º.  
**–ü—Ä–∏–º–µ—Ä**:  
- –ú–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ —Ü–µ–Ω—ã –∞–∫—Ü–∏–∏:  
\[ S_t = S_0 \times e^{(Œº - \frac{œÉ^2}{2})t + œÉW_t} \]  
–≥–¥–µ \(W_t\) ‚Äî –≤–∏–Ω–µ—Ä–æ–≤—Å–∫–∏–π –ø—Ä–æ—Ü–µ—Å—Å.  

**–ö–æ–¥**:  
```python
import numpy as np
import matplotlib.pyplot as plt

# –ü–∞—Ä–∞–º–µ—Ç—Ä—ã –º–æ–¥–µ–ª–∏
n_steps = 1000
t = np.linspace(0, 1, n_steps)
mu = 0.1    # –°—Ä–µ–¥–Ω—è—è –¥–æ—Ö–æ–¥–Ω–æ—Å—Ç—å
sigma = 0.2 # –í–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å
S0 = 100    # –ù–∞—á–∞–ª—å–Ω–∞—è —Ü–µ–Ω–∞

# –ì–µ–Ω–µ—Ä–∞—Ü–∏—è –±—Ä–æ—É–Ω–æ–≤—Å–∫–æ–≥–æ –¥–≤–∏–∂–µ–Ω–∏—è
dW = np.random.normal(0, np.sqrt(1/n_steps), n_steps)
W = np.cumsum(dW)
S = S0 * np.exp((mu - 0.5*sigma**2)*t + sigma*W)

# –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è
plt.plot(t, S)
plt.title("–ì–µ–æ–º–µ—Ç—Ä–∏—á–µ—Å–∫–æ–µ –±—Ä–æ—É–Ω–æ–≤—Å–∫–æ–µ –¥–≤–∏–∂–µ–Ω–∏–µ")
plt.xlabel("–í—Ä–µ–º—è")
plt.ylabel("–¶–µ–Ω–∞ –∞–∫—Ü–∏–∏")
plt.show()
```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –ò–∑–º–µ–Ω–∏—Ç–µ –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å (œÉ) –Ω–∞ 0.3. –ö–∞–∫ –º–µ–Ω—è–µ—Ç—Å—è –≥—Ä–∞—Ñ–∏–∫?  
- **–û—Ç–≤–µ—Ç**: –ì—Ä–∞—Ñ–∏–∫ —Å—Ç–∞–Ω–µ—Ç –±–æ–ª–µ–µ ¬´—Ä–≤–∞–Ω—ã–º¬ª –∏–∑-–∑–∞ —É–≤–µ–ª–∏—á–µ–Ω–∏—è —Å–ª—É—á–∞–π–Ω—ã—Ö –∫–æ–ª–µ–±–∞–Ω–∏–π.

---

**–®–∞–≥ 2: –ü—É–∞—Å—Å–æ–Ω–æ–≤—Å–∫–∏–π –ø—Ä–æ—Ü–µ—Å—Å**  
**–ü–æ–Ω—è—Ç–∏–µ**: –ú–æ–¥–µ–ª–∏—Ä—É–µ—Ç —Ä–µ–¥–∫–∏–µ —Å–æ–±—ã—Ç–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∫–ª–∏–∫–æ–≤ –Ω–∞ —Å–∞–π—Ç –∑–∞ —á–∞—Å).  
**–ü—Ä–∏–º–µ—Ä**:  
- –°—Ä–µ–¥–Ω–µ–µ —á–∏—Å–ª–æ –∫–ª–∏–∫–æ–≤ –≤ —á–∞—Å: 5. –ö–∞–∫–æ–≤–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å 3 –∫–ª–∏–∫–æ–≤ –∑–∞ —á–∞—Å?  
\[ P(X=3) = \frac{5^3 e^{-5}}{3!} ‚âà 0.1404. \]  

**–ö–æ–¥**:  
```python

```

**–ü—Ä–∞–∫—Ç–∏–∫–∞**:  
- –î–ª—è Œª=10, –Ω–∞–π–¥–∏—Ç–µ P(X=7). (–û—Ç–≤–µ—Ç: ‚âà 0.0901).

---

### **–£—Ä–æ–∫ 6.2: –ë–∞–π–µ—Å–æ–≤—Å–∫–∏–µ —Å–µ—Ç–∏ ‚Äî –º–æ–¥–µ–ª–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–ª–æ–∂–Ω—ã—Ö –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–µ–π**  
**–¶–µ–ª—å**: –ù–∞—É—á–∏—Ç—å—Å—è –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—Ç—å –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –º–µ–∂–¥—É –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–º–∏.  

---

**–®–∞–≥ 1: –ü—Ä–∏–º–µ—Ä –º–µ–¥–∏—Ü–∏–Ω—Å–∫–æ–π –¥–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∏**  
- –£–∑–ª—ã: –ë–æ–ª–µ–∑–Ω—å (D), –°–∏–º–ø—Ç–æ–º (S).  
- –°–≤—è–∑–∏: D ‚Üí S.  

**–ö–æ–¥**:  
```python

from pgmpy.factors.discrete import TabularCPD

# –°–æ–∑–¥–∞–Ω–∏–µ –º–æ–¥–µ–ª–∏

# –û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —É—Å–ª–æ–≤–Ω—ã—Ö –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π
cpd_d = TabularCPD(variable='D', variable_card=2, values=[[0.99], [0.01]])  # 1% –±–æ–ª–µ–∑–Ω—å

    variable='S',
```
