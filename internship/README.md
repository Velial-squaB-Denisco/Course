#### b) **SQLAlchemy ORM** ‚Äî "–í–æ–ª—à–µ–±–Ω—ã–π –ø–µ—Ä–µ–≤–æ–¥—á–∏–∫"
- –í—ã—Å–æ–∫–æ—É—Ä–æ–≤–Ω–µ–≤—ã–π –ø–æ–¥—Ö–æ–¥. –¢—ã —Ä–∞–±–æ—Ç–∞–µ—à—å —Å –¥–∞–Ω–Ω—ã–º–∏ –∫–∞–∫ —Å –æ–±—ä–µ–∫—Ç–∞–º–∏ Python.

### 4. **–û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è –ø—Ä–æ—Å—Ç—ã–º–∏ —Å–ª–æ–≤–∞–º–∏:**

- –ö–∞–∂–¥–∞—è —Å—Ç—Ä–æ–∫–∞ ‚Äî —ç—Ç–æ –æ–±—ä–µ–∫—Ç `Toy`.
- –ß—Ç–æ–±—ã –¥–æ–±–∞–≤–∏—Ç—å –Ω–æ–≤—É—é –∏–≥—Ä—É—à–∫—É, —Ç—ã –ø—Ä–æ—Å—Ç–æ —Å–æ–∑–¥–∞–µ—à—å –æ–±—ä–µ–∫—Ç `Toy(name="–ö—É–∫–ª–∞", color="—Ä–æ–∑–æ–≤–∞—è")` –∏ –≥–æ–≤–æ—Ä–∏—à—å —Å–µ—Å—Å–∏–∏: ¬´–ó–∞–ø–æ–º–Ω–∏ —ç—Ç–æ!¬ª.
